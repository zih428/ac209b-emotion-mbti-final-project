## Naming Conventions

- Markdown files are the default source for agent reading.
- `artifacts/submitted/` contains original submitted notebooks/slides.
- `artifacts/drafts/` contains earlier iterations or possible auto-saved versions.
- `artifacts/notes/` contains original binary notes such as DOCX files.
- Requirement screenshots are kept in `artifacts/`; requirement content is summarized in `requirements.md`.
