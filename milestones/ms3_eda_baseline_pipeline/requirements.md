## Notebook Guidelines

- Start with a clear problem statement.
- Organize logically with headings and subheadings.
- Include a table of contents.
- Explain code, choices, and results using comments and Markdown.
- Use clear, well-labeled visualizations with descriptive captions.
- Keep code readable and reproducible.
- Notebook should run error- and warning-free if reproducibility is needed.

## Slide Tips

- Make plots readable.
- Label everything clearly.
- Use fewer words.
- One idea per slide.
- Add slide numbers.
- Slides should tell a clear story, not a collection of disconnected results.
